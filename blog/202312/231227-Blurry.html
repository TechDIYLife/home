拉普拉斯算法（Laplacian）：

通过使用Laplacian算子，求输入图片的二阶导图。二阶倒数的值变化大的地方，就是图像中物体的边缘信息。通过对二阶导图求方差，根据方差值的大小可以判断出图像模糊的模糊程度。方差值越小，图像越模糊。

比如下面的两种图片，模糊图片的Blurry值明显比清晰图片的值小很多。


拉普拉斯算法Blurry值示意图

此方法的问题：
在不同场景中，清晰与模糊之间的阈值变化很大，因此，难以确定一个固定的阈值来处理多种场景。

傅里叶变换算法：

另一个算法是，基于傅立叶变换(FFT)的图片模糊检测算法。
此算法，首先将输入图片转变为频谱图。频谱图中，中心代表的是低频 （下图中第二列），往四面八方扩展后逐渐变为高频。通过屏蔽掉频谱图的中心区域，对图像实现高通滤波，保留图像中边缘等高频的信息。然后对频谱图求均值，求图片中平均高频幅值，通过该平均幅值来判断图像是否模糊。与上面一样，平均幅值越小，图像越模糊。


FFT傅里叶变换算法示意图

此图来自：http://luohanjie.com/2023-08-18/blur-detection-with-fft.html

1.2 算法验证

相比拉普拉斯算法，基于FFT的算法的参数更多，返回blurry值的范围与拉普拉斯相似，不同场景下值的范围也不同。根据，我初步的评价，拉普拉斯算法应该是可以满足我目前的需求。
下面是此算法的代码，以及验证的结果。

拉普拉斯算法
安装开发库：

pip install imutils

程序代码：

from imutils import paths
import argparse
import cv2
def variance_of_laplacian(image):
  return cv2.Laplacian(image, cv2.CV_64F).var()

for i in range(1,12):
    image = cv2.imread("tmp1/frame"+str(i)+".jpg")
    image = cv2.imread("img1/"+str(i)+".jpg")
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    fm = variance_of_laplacian(gray)
    detect="NA"
    if fm >30: detect="OK"
    print(i, fm, detect)
