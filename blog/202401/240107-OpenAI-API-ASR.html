语音识别：如何使用OpenAI API处理语音转文本

三分钟视频演示

1 OpenAI账号注册与登录

OpenAI API服务的管理入口网址（Platform）：
https://platform.openai.com/login

如果没有注册，可以先进行用户注册。


图1. 用户登录界面，如果没有注册，点击注册进行用户注册

2 账号充值

使用API服务，需要先进行充值。
登录Platform之后，按照图2，图3，进行充值。
充值前需要添加信用卡等支付方式（图3-1）。

充值金额最低为5美金。


图2：账号充值入口


图3：账号充值（1. 添加支付方式； 2. 设置充值金额）

注意：建议设置费用配额，以避免产生超出预期的高额费用。
设置方式：Setting --> Limits (Usage limits)

3 创建API Key

您需要创建API Key来访问OpenAI的API服务。创建方式参考图4。
创建完成之后，要注意保管API Key。
建议配置到系统环境变量中，不要写在代码中，以免泄露。


图4：创建API Key

4 安装开发库 

pip install openai

5 调用代码

STT API的特点：
只接收语音文件，不支持实时语音数据传输。
指定语言可以提高识别精度，并减少响应速度，需要根据需求来确定。

import os
from openai import OpenAI

## 在Python中设置
os.environ['OPENAI_API_KEY'] = "your_api_key_here"

client = OpenAI()

audio_file= open("/path/to/file/audio.mp3", "rb")
transcript = client.audio.transcriptions.create(
  model="whisper-1", 
  file=audio_file
)

# 默认返回值transcript为Transcription对象，访问方式如下
print(transcript.text)
