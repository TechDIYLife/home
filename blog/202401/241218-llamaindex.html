2 什么是LlamaIndex？

LlamaIndex is a framework for building context-augmented generative AI applications with LLMs including agents and workflows.

  • 常见用例包括：
    • 问答系统（检索增强生成，简称 RAG）
    • 聊天机器人
    • 文档理解与数据提取
    • 能够执行研究和采取行动的自主智能体
    • 结合文本、图像和其他数据类型的多模态应用
    • 对数据进行微调模型以提升性能

3 LlamaIndex 适合谁？

LlamaIndex是面向AI技术开发者的一款功能，它可以满足从初学者的快速入门到高级用户的自定义和扩展模块的各种需求。

  • 高阶 API：初学者可以使用 LlamaIndex 提供的高级API，通过 5 行代码即可导入并查询数据。
  • 底层 API：对于更复杂的应用，高级用户可以自定义和扩展任何模块，包括数据连接器、索引、检索器、查询引擎和重排序模块，以满足特定需求。

4 几个关键词

  • 什么是上下文增强？
    上下文增强（Context Augmentation）是指在处理任务时，通过增加或丰富上下文信息来提高模型的理解能力和性能的方法。
    LLMs 虽然在大量公开数据上进行了预训练，但这些模型并没有训练过你的数据。你的数据可能是私有的，或者与要解决的问题特定相关。这些数据可能存在于 API 后端、SQL 数据库中，或被困在 PDF 文件和幻灯片中。

    上下文增强最流行的示例就是 RAG，检索增强生成（Retrieval-Augmented Generation）

  • 什么是智能体？
    智能体是由 LLM 驱动的知识助理，它们能够使用工具执行任务，如研究、数据提取等。复杂的智能体不仅可以回答问题，还可以具备感知、决策，以及采取行动的能力，以完成更复杂的任务。

  • 什么是工作流？
    工作流是包含多种功能的，比如多个智能体，多种数据连接器，以事件驱动的多步流程。
    可以完成执行各种复杂任务，同时具备反思、错误修正等先进 LLM 应用的特性。

参考资料：https://docs.llamaindex.ai/en/latest/
