1 内容概要

  • 介绍如何在Linux系统上安装 Jupyter Notebook。
  • 介绍如何在Jupyter中使用GPU

2 开发环境

  • Ubuntu 20.04
  • Python 3.10
  • GPU RTX 3090

3 为什么安装Jupyter？

在学习，或者调试一些简单的代码时，经常使用Colab，不仅免费方便易用，还不需要维护硬件。
但是最近Colab免费版，最近越来越难以申请到GPU资源了。

安装Jupyter可以作为替代谷歌Colab的方案，用于个人学习，或者简单的实验用。

4 如何安装？

安装过程可以分为三步，分别是设置运行环境，安装pytorch，安装jupyter。
在安装以前，建议先对Linux系统进行升级：sudo apt -y upgrade

  • 第一步：新建Conda运行环境

    #设置CUDA
export CUDA_HOME=/usr/local/cuda-12
export LD_LIBRARY_PATH=$CUDA_HOME"/lib64:$LD_LIBRARY_PATH"
export PATH=$CUDA_HOME"/bin:$PATH"

#创建运行环境
conda create -n jupyter python=3.10
conda activate jupyter

  • 第二步：安装pytorch相关的库 (做AI必备工具)

    conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia

  • 第三步：安装jupyter

    pip3 install Jupyter
pip3 install jupyterlab

5 如何使用？

  • 启动

    # 本机访问
jupyter-lab
# 支持网络访问
jupyter-lab --ip='0.0.0.0'

    启动成功后，会提示访问的链接。

  • 打开浏览器，访问
    本机访问地址类似： http://127.0.0.1:8888/lab?token=xxxxxx
    通过网络访问：http://servername:8888/lab?token=xxxxxx

6 如何使用GPU？

如果，你发现jupyter不使用GPU进行计算，你可以按照下面的6.1到6.2的说明来解决。

6.1 事前检查

  1. 安装Pytorch的GPU版本

    #参考安装命令
conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia

  2. 执行下面的代码，检查是否支持GPU

    import torch
torch.cuda.is_available()

6.2 Pytorch默认使用CPU，使用GPU需要指定

在PyTorch的设置中，如果代码中没有明确指示使用 GPU（例如，通过 .to('cuda') 将模型和数据移动到 GPU），PyTorch 默认会使用 CPU 执行计算。

6.2.1 使用GPU的方法一：代码显式指定GPU 

from transformers import AutoTokenizer, AutoModelForCausalLM

tokenizer = AutoTokenizer.from_pretrained("google/gemma-2b-it")
model = AutoModelForCausalLM.from_pretrained("google/gemma-2b-it")

input_text = "Write me a poem about Machine Learning."
input_ids = tokenizer(input_text, return_tensors="pt")

model = model.to('cuda')
input_ids = input_ids.to('cuda')

outputs = model.generate(**input_ids)
print(tokenizer.decode(outputs[0]))

6.2.2 使用GPU的方法二：改变Pytorch默认设置 

import torch
torch.set_default_device('cuda')

7 总结

个人感觉便利性上比Colab略差，比如：

  • 执行的按钮，在最顶部，colab的在cell左边方便直观。
  • 执行的状态没有显示
  • GPU显存占用后，不会主动释放，需要重启kernel

优点就是自己的机器，想怎么用就怎么用。
