微调技术大比拼：全量微调与LoRA、QLoRA实测对比！

内容概要

  • 实测对比：全量微调、LoRA微调、QLoRA微调的GPU显存占用和训练速度。
  • 微调案例演示：如何微调一个产品问答AI小助手

B站视频演示：https://www.bilibili.com/video/BV1TTbFegELr/

1 环境准备

1.1 服务器登录与基本设置

  • 登录服务器并租用一个新实例
    https://gpuez.com

    租用实例入口：控制台（右上角） --> 算力（左侧）--> 租用新实例

1.2 基本环境安装

  • 基本环境
conda init bash
. ~/.bashrc
git init
sudo apt update
sudo apt install less nano
  • 配置cuda
export CUDA_HOME=/usr/local/cuda
export LD_LIBRARY_PATH=$CUDA_HOME"/lib64:$LD_LIBRARY_PATH"
export PATH=$CUDA_HOME"/bin:$PATH"
  • 设置conda的文件下载镜像

创建文件

nano ~/.condarc

文件内容如下：

channels:
  - defaults
show_channel_urls: true
default_channels:
  - https://mirrors.sustech.edu.cn/anaconda/pkgs/main
  - https://mirrors.sustech.edu.cn/anaconda/pkgs/free
  - https://mirrors.sustech.edu.cn/anaconda/pkgs/r
  - https://mirrors.sustech.edu.cn/anaconda/pkgs/pro
  - https://mirrors.sustech.edu.cn/anaconda/pkgs/msys2
custom_channels:
  conda-forge: https://mirrors.sustech.edu.cn/anaconda/cloud
  msys2: https://mirrors.sustech.edu.cn/anaconda/cloud
  bioconda: https://mirrors.sustech.edu.cn/anaconda/cloud
  menpo: https://mirrors.sustech.edu.cn/anaconda/cloud
  pytorch: https://mirrors.sustech.edu.cn/anaconda/cloud
  simpleitk: https://mirrors.sustech.edu.cn/anaconda/cloud
  nvidia: https://mirrors.sustech.edu.cn/anaconda-extra/cloud
  • 安装python运行环境
conda create -n test1 python=3.10
conda activate test1
export PATH=~/miniconda3/envs/test1/bin:$PATH
  • 安装pytorch
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia
  • 安全其他相关的库
pip install modelscope
pip install bitsandbytes
pip install deepspeed==0.14.4

1.3 安装 llama factory 

git clone --depth 1 https://github.com/hiyouga/LLaMA-Factory.git
cd LLaMA-Factory
pip install -e ".[torch,metrics]"

设置从魔搭下载模型

export USE_MODELSCOPE_HUB=1

2 模型微调

打开文件 examples/train_lora/llama3_lora_sft.yaml，将模型名称修改如下：

model_name_or_path: LLM-Research/Meta-Llama-3-8B-Instruct

使用以下命令开启模型微调：

llamafactory-cli examples/train_lora/llama3_lora_sft.yaml

3 模型测试

打开 examples/inference/llama3_lora_sft.yaml 文件，修改基础模型和微调模型名称：

model_name_or_path: LLM-Research/Meta-Llama-3-8B-Instruct
adapter_name_or_path: saves/llama3-8b/lora/sft
template: llama3
finetuning_type: lora

在命令行中执行一下命令：

llamafactory-cli chat examples/inference/llama3_lora_sft.yaml

量化技术对比

微调模型：Meta-Llama-3-8B-Instruct

4 AI小助手案例实战

4.1 数据准备

  • 下载数据：https://techdiylife.github.io/blog/202401/gpuez.json
  • 将gpuez.json文件上传到llama-factory/data目录下

4.2 数据准备代码

  • 在 dataset_info.json 中增加数据描述

    "gpuez": {
  "file_name": "gpuez.json",
  "formatting": "sharegpt",
  "columns": {
    "messages": "conversations"
  }
},

  • 打开 examples/train_lora/llama3_lora_sft.yaml 修改dataset名称

    dataset: gpuez

4.3 模型训练与测试

模型微调与测试

llamafactory-cli train examples/train_lora/llama3_lora_sft.yaml
llamafactory-cli chat examples/inference/llama3_lora_sft.yaml
